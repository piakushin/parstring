# Parstring
Simple parser of "encoded" mathematical expressions.

## Usage
### 0. Install rustc & cargo
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
### 1. Checkout this repo
```bash
gh repo clone piakushin/parstring
```
### 2. Run
Run application:

```bash
cargo run --release -- <expression>
```

### 3. Output example
```bash
2a2 = 4
```
